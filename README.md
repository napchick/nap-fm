# nap.fm - персональный сервис аудиоаналитики

При помощи данного инструмента можно отслеживать свою историю прослушивания, а также получить различную аналитику, например топ 5 самых прослушиваемых песен за разные периоды, количество прослушиваний у каждого из артистов, распределение прослушиваний по странам и так далее.
В настоящий момент проект еще не завершен, однако большинство основных функций уже реализовано. Изначально данный проект создавался сугубо для меня, по сей день все так и остается, однако в будущем рассматривается возможность сделать его многопользовательским(хотя бы для нейскольких человек)

Посмотреть на работу сервиса можно по [ссылке](https://nap-fm-h9hgx5da7tu2qdbteig9h2.streamlit.app/) 

# Возможности сервиса

На данный момент nap.fm состоит из 3 страниц: overview, library и reports

- ## overview:
  Это главная страница, на которой можно вкратце увидеть основную информацию, а именно:
  - общее количество прослушиваний
  - количество уникальных артистов
  - 10 последних прослушанных песен (по названию песни или артиста можно перейти на их страницу, где будет различная информация о песне/артисте)
  - 5 самых прослушанных песен с возможностью выбора периода
  - 5 самых прослушаннх песен с возможностью выбора периода
    
- ## library:
  На данной странице же можно получить уже более полную информацию:
  - полная история прослушиввания с указанием даты и времени
  - количество прослушиваний у каждого артиста
  - количество прослушиваний у каждой песни

  В ближайшее врему будет добавлена возможность выбора периода, чтобы можно было получить агрегированную информацию не за все время, а за интересующий временной промежуток

- ## reports:
  На этой странице в основном представлены различные распределения:
  - количество прослушанных песен из различных десятилетий, также можно узнать самую прослушиваемую песню из каждого десятилетия (можно выбрать период)
  - распределение прослушиваний по часам в течение дня (можно выбрать период)
  - распределение числа исполнителей из различных стран
  - распределение числа прослушанныз песен из различных стран
