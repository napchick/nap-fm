# nap.fm - персональный сервис аудиоаналитики

При помощи данного инструмента можно отслеживать свою историю прослушивания, а также получить различную аналитику, например топ 5 самых прослушиваемых песен за разные периоды, количество прослушиваний у каждого из артистов, распределение прослушиваний по странам и так далее.
В настоящий момент проект еще не завершен, однако большинство основных функций уже реализовано. Изначально данный проект создавался сугубо для меня, по сей день все так и остается, однако в будущем рассматривается возможность сделать его многопользовательским(хотя бы для нейскольких человек)

Посмотреть на работу сервиса можно по [ссылке](https://nap-fm-h9hgx5da7tu2qdbteig9h2.streamlit.app/)
